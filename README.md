# my-context

きょうすけとAIが共同で使うコンテキストリポジトリ。

AIとの会話にこのリポジトリを読み込ませることで、毎回ゼロから説明しなくてもよい状態を維持する。

## 構成

| フォルダ | 内容 |
|---------|------|
| `00_personal/` | きょうすけ自身のプロフィール・目標・スタイル |
| `10_shared/` | 全事業で使う共通資産（スキル・リンク） |
| `20_businesses/` | 事業ごとの詳細 |
| `30_ideas/` | アイデア置き場 |
| `99_archive/` | 撤退・終了したものの保管庫 |

## 使い方

- 新しいAIセッションの冒頭にこのリポジトリを渡す
- 情報が変わったら該当ファイルを更新してコミット・プッシュ
- 新しい事業を始めたら `20_businesses/biz_X_xxx/` を追加
