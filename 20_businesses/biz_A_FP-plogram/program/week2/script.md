## 原稿

スライドNo	スクリプト
1	こんにちは、Week2へようこそ！ / Week1.いかがだったでしょうか。家計簿アプリの入力、はじめは面倒ですが習慣にしてしまえばこっちのものです。とても大事なのでまずはこのプログラム中頑張って継続していきましょう！Week1では、第1の生活費口座、第2の生活防衛費口座、そして第3の投資口座、という3口座管理法についてお伝えしました。この仕組み、シンプルですがめちゃくちゃ効果的です。 そして今週、Week2では、この3口座管理法の土台となる生活費口座の部分を作り始めていきます。
2	Week2の目的はずばり、「支出の正確な洗い出し」、そしてそこから「本当の支出と収入を比べて、現状をしっかり把握すること」です。 / Week1で学んだ3口座管理法を機能させるためには、「第1口座（生活費口座）に、本当はいくら必要なのか？」を正確に知る必要があります。
3	でも、多くの人はここを「だいたい月30万くらいかな…」と"なんとなく"で済ませています。これじゃダメなんです。 / なぜか？"なんとなく"で済ませると、必ず見落としがあるからです。そして、その見落としが積み重なって「毎月お金が足りない…」「貯金したいのに、気づいたら残ってない…」という状況が生まれてしまいます。そこで、正確に支出を洗い出すための2つの重要な視点をお伝えします。
4	視点1つ目は、すべての支払いの「月平均化」。これがWeek2の核心です。 / 毎月かかるもの、年に1回のもの、2年に1回のもの。すべてを「月平均」に直して、月に本当はいくら必要なのかを明確にします。 例えば、車の費用。「毎月のローンが28,000円、ガソリンが1万ちょっとだから、まあ月4万円くらいかな」そう思っていませんか？ / でも、実際は違います。 / 年に1回の自動車税、自動車保険、2年に1回の車検……。 / これらを全部足して「12」で割ると、なんと月平均「73,000円」かかっていたんです。 / この「見えていない3万円」が、家計を圧迫している真犯人です。こういった見えていない支出を、Week2ではもれなくあぶり出していきます。
5	"そして、もう一つ大事な考え方をお伝えします。「未来の消耗品への備え」についてです。 / 家具、家電、そして自動車など。これらは資産ではなく、「数年単位で必ず買い替える消耗品」と考えてください。 / 壊れてから慌てて貯金を取り崩すのではなく、あらかじめ「予算」に組み込みます。
 例えば20万円の洗濯機を10年で買い替える場合。月割りすると、月々約1,700円になります。この金額を家電積立として計上し、毎月積み立てておきます。これをやっていないと、モノが壊れた瞬間に家計が破綻してしまいます。 /このようにおおまかでもいいので各消耗品の「買い替えまでの期間」と「概算金額」を出して、月割り計算していきます。 "
6	Week2では、この「見えない支出」や「未来の消耗品」も含めて、支出を7つのカテゴリに分けて徹底的に洗い出します。 / 1.住宅関連費、2.車関係費、3.保険、4.通信、5.生活費、6.教育・育児費用、7.贅沢費。 / これらに分けて、毎月、年1回、数年に1回、未来の買い替えなど、想定しうる支出を全て洗い出し、月平均に直します。そうすることで、「生活費口座に本当に月いくら必要なのか」が見えてきます。
7	"支出が出たら、後半は収入との比較です。 / 給与やボーナス、公的手当などを正確に把握して、「年間収入から年間支出を引いた収支ギャップ」を出します。これがプラスなのかマイナスなのか、具体的にいくらなのか。これを明確にします。 / ここで初めて、家計の本当の姿が見えてきます。
 / 「毎月いくら黒字なのか」ということが浮き彫りになり、同時に、「じゃあ、この金額を貯蓄や投資に回せるな」と、ゴールが見えてくるんです。これが未来を変える第一歩です！"
	"支出洗い出し表
・支払い頻度、金額の入力方法　各項目の詳細はガイドを参照
・項目がない、足りないときの追加方法
・生活費、贅沢費の細かい項目の考え方
収入洗い出し表
収支表"
8	それでは、まとめです。今週のワークは、収支洗い出しシートを埋めることです。 / 地道な作業ですが、これがあなたの未来を変える第一歩です。わからないことがあれば、LINEで質問してください。一緒に進めていきましょう！週末面談では、この結果について詳しく伺っていきたいと思います！ / では、今週も一緒に頑張っていきましょう！お疲れさまでした
	"では、実際の画面を見てみましょう。 / ファイルを開くと、このような表が出てきます。 このように７つのカテゴリに沿って細かく項目が入力されています。これら一つ一つの金額欄と支払い頻度の欄を入力していきます。例えば、自動車税。年一回の支出で、36000円、と入力すると。月平均のところに3000円と自動で入力されます。このように、不定期の支出を全て月平均に慣らしていく作業をしていきます。
固定費なんかは金額がはっきりしているので分かりやすいと思いますが、食費や日用品など、はっきり分からない支出も多いと思います。そういうのはとりあえずだいたいこれくらいかな？という金額を入れてみてください。これらは家計簿アプリで日々の記録をためているので、記録が貯まったら見返しながら調整していきましょう。まずは入力して最後まで埋めましょう！細かい各項目の考え方はノートにガイドを載せているので、確認しながら入力してみてください。
支出がすべて入力できたら、自動で一番下に合計の月間支出額、年間支出額が集計されます。そしたら、その横の「収入洗い出しシート」も入力しましょう。こちらは項目が少ないのですぐに入れられると思います。毎月の給料や、ボーナス、公的手当などを入力していきます。そうすると、こちらも一番下に月間収入額、年間収入額が出ます。年間支出と年間収入を比べてどうだったか、結果が聞けるのを楽しみにしてますね"
