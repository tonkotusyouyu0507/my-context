## Canvaファイル構成（確定）

1ファイルの中に全ページを収める。毎回コピーして使う。

```
📄 型テンプレ.canva（コピー元・直接編集禁止）
　├── ページ1：表紙型 ✅（完成済み 2026-05-24）
　├── ページ2：内容・テキストメイン型（作成中）
　├── ページ3：内容・比較型（作成中）
　└── ページ4：CTA型（LINE誘導）

📁 インスタ投稿フォルダ（Canva）
　├── 型テンプレ ← 触らない
　├── 投稿①_テーマ_日付
　├── 投稿②_テーマ_日付
　└── ...
```

**毎回の作業手順：**
1. 型テンプレを右クリック →「コピーを作成」
2. ファイル名を「投稿①_テーマ_日付」に変更
3. テキストだけ変える（写真・色・フォントは触らない）
4. PNGでダウンロード → インスタに投稿

## デザインテンプレート（画像ベース 4型）

### 基本仕様

| 項目 | 値 |
|---|---|
| サイズ | 1080 × 1350px（4:5縦型） |
| ベース | 写真を全面に使う（フルブリード） |
| テキスト色 | オフホワイト `#FAF7F2`（写真に重ねる場合） |

| フォント（メイン） | **Noto Sans JP**（Canvaで検索：「Noto Sans JP」）|
| フォント（代替） | **Noto Sans JP**（代替なし・これ一択） |
| フォントウェイト | 見出し：Bold / 本文：Regular |
| アカウント名 | 全スライドの右下か左下に小さく入れる |

**Canvaでのフォント設定：**
1. テキストボックスを選択
2. フォント検索欄に「M PLUS」と入力
3. 「M PLUS Rounded 1c」を選択 → これが丸ゴシック

### Type A｜フック型（1枚目）：写真フルブリード＋テキストオーバーレイ

**目的：スクロールを止める。**

```
┌─────────────────────────┐
│                         │
│  [写真フルブリード]      │ ← スマホ・財布・お金などの写真
│                         │
│  ╔═══════════════════╗   │
│  ║ ＼仕組み化家計｜  ║   │ ← 半透明ピルバッジ（白or暗色）
│  ║   きょうすけ  ／  ║   │
│  ╚═══════════════════╝   │
│                         │
│  貯金できないのは       │ ← 白・超太字・大きく（ドロップシャドウあり）
│  意志の問題じゃない     │
│                         │
│             @kyosuke 📌  │ ← 右下に小さく
└─────────────────────────┘
```

**ポイント：**
- 写真の暗い部分にテキストを置く（読みやすい）
- または写真下半分に半透明黒オーバーレイ（グラデーション）を敷く
- テキストにドロップシャドウを追加（写真の上でも読めるように）

**Canvaでの作り方：**
1. 写真をフルブリードで配置
2. 写真下半分に「暗めのグラデーション矩形」を半透明で重ねる（透明度50〜70%）
3. その上に白太字テキストを配置

## 写真を撮るときのコツ（スマホで撮影OK）

### 📌 写真選定の3原則（確定）

| # | 基準 | 詳細 |
|---|---|---|
| ① | **俯瞰アングル（真上から）** | カメラを真上に向けて撮る。斜めは使わない |
| ② | **自然光・暗めの色温度** | 窓際の光。暖色系・落ち着いたトーンで統一 |
| ③ | **小物は2〜3点まで** | ごちゃつかせない。引き算で決める |

### おすすめの撮影シーン

| シーン | 使う小物（2〜3点） | 使いどき |
|---|---|---|
| **朝のデスク** | コーヒー・手帳・ペン | 日常系・共感系投稿 |
| **お金の俯瞰** | お札・財布 | 実績・家計系投稿 |
| **アプリ画面** | スマホ・コーヒー | NISA・資産系投稿 |
| **ノートの書き込み** | ノート・ペン | 仕組み化・ノウハウ系 |
| **財布＋カード** | 財布・カード1枚 | 節約・支出管理系 |

### 撮影チェックリスト（毎回確認）

- [ ] 真上から撮っているか
- [ ] 自然光が当たっているか（窓際か）
- [ ] 小物が3点以内か
- [ ] 背景がシンプルか（白い机・木目テーブル）
- [ ] 色温度が暗め・落ち着いているか

## NG集（画像ベース版）

- ❌ 写真にそのままテキストを重ねる（読みにくい）→ グラデーションオーバーレイを敷く
- ❌ テキストを細いフォントにする → Bold以上必須
- ❌ 写真とテキストが喧嘩する → コントラストを確認
- ❌ アカウント名を入れ忘れる → 保存・シェアされても誰か分からなくなる
- ❌ スライドごとに写真のトーンがバラバラ → 同じ色温度で統一
- ❌ 情報を詰め込む → 1スライド1メッセージ
