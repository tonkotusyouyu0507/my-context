### 競合に共通するパターン（黄金則）

1. **コンテキストバッジ**：ピル型の小さなラベルを上部に配置「＼年200万貯めるママ／」
2. **超大文字**：1スライドに1メッセージ、文字は画面の6〜7割を占める
3. **高コントラスト**：暗い背景に白、または明るい背景に黒
4. **アカウント名を画像内に入れる**：保存・シェアされても誰の投稿か分かる
5. **1枚目で結論を予告する**：「知らないと損」「〇〇した結果」

### きょうすけの差別化ポイント

- **ターコイズ単色**は競合に一切いない → 色だけで即識別できる強みになる
- **男性×家計管理×仕組み化**の組み合わせは市場にない
- 現状の課題：テキストが細い・バッジなし・インパクトが弱い

### Type A｜フック型（1枚目専用）

**目的：スクロールを止める。続きが読みたくなる。**

#### A-1：ターコイズ爆発型（強インパクト）

```

レイアウト：
┌─────────────────────────┐
│                         │
│  ＼仕組み化家計  /      │ ← コンテキストバッジ（白枠・小さめ）
│                         │
│                         │
│   NISAより先に          │ ← メイン見出し（白・超極太・80〜100px相当）
│                         │
│   やるべきことが        │ ← サブ（白・中太・40〜50px）
│   ある                  │
│                         │
│                         │
│              kyosuke 📌  │ ← アカウント名（右下・小さく白）
└─────────────────────────┘

カラー：背景 #0D6963 / テキスト #FFFFFF
フォント：Noto Sans JP ExtraBold or Black
ポイント：テキストを左揃えにする（中央揃えより読みやすい）
```

**使いどき：** ノウハウ系・主張系投稿の1枚目

## Canvaでの作り方（手順）

### 共通設定

1. カスタムサイズ：`1080 × 1350px`
2. ブランドカラーを登録：`#0D6963` / `#20B2AA` / `#1AA39B` / `#C9A028`
3. フォント：**Noto Sans JP ExtraBold**（見出し）＋ **Noto Sans JP Regular**（本文）

### アカウント名の入れ方

- 右下に固定テキスト：`kyosuke`（または `きょうすけ`）
- サイズ：16〜18px
- カラー：白（ダーク背景）または ターコイズ（白背景）
- 全スライドに入れる（保存・シェア対策）

### コンテキストバッジの作り方

- テキスト：`＼仕組み化家計 ／`
- 背景：白（ターコイズ背景のとき）または ターコイズ（白背景のとき）
- 角丸：50px（ピル型）
- パディング：8px 16px
- サイズ：12〜14px

## デザインNG集

- ❌ 文字が細い（Regular / Light フォント）→ 必ず Bold 以上
- ❌ 1スライドに情報を詰め込みすぎる → 1枚1メッセージ
- ❌ アカウント名を入れ忘れる → 保存されても誰か分からなくなる
- ❌ 色を多用する → ターコイズ・白・黒の3色で統一
- ❌ フォントをバラバラにする → Noto Sans JP 1本で統一
- ❌ グラデーション使用 → ブランドルール違反（ソリッドカラーのみ）
