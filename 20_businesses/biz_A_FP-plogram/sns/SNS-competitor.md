## 市場全体で見えたパターン

### ① 競合の男女比

- **#家計管理 上位は女性クリエイターがほぼ独占**（ターゲットは主婦・ママ層）
- **男性クリエイターはNISA・投資寄りに数人**（家計管理×仕組み化の男性はほぼいない）
- → きょうすけの「男性×家計管理×仕組み化×ADHDパパ」は未開拓ニッチ

### ② プロフィールの黄金型

```
1行目：数字入り実績「NISAで+200万」「年200万貯める」「資産700万達成」
2行目：自分の属性・信頼性「26歳・投資歴6年」「元浪費家ママ」「ズボラ主婦」
3行目：発信テーマ or 読み手へのメッセージ
最後 ：CTA（LINE・リンク）
```

### ③ 投稿形式トレンド

| 形式 | 特性 | 位置づけ |
|---|---|---|
| リール（縦型動画） | リーチ広い・バズりやすい | 認知獲得・新規流入がメイン |
| カルーセル | 保存されやすい・じっくり読まれる | フォロワーへの価値提供がメイン |

→ 上位勢はリール主体。カルーセルは「補助」として使っている。
→ **二刀流（リール＋カルーセル）が現実解**

### ④ バズるフックの型

| 型 | 例 |
|---|---|
| 危機感型 | 「9割が知らない…」「知らないと損」 |
| 体験談型 | 「20歳でNISAはじめたら増えすぎてこうなった」「ニーサ5年目」 |
| 権威×具体型 | 「年200万貯めるママがやっていること」 |
| 比較・答え合わせ型 | 「オルカン vs S&P500やった結果」「現金 vs カード」 |
| 条件制限型 | 「手取り17万でも年80万貯めた」「1馬力でも100万貯めたママ」 |
